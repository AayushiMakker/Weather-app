
# 🌦 Weather Dashboard Web App

This project is a simple weather dashboard web application that allows users to search for weather information for different cities. The application displays temperature, city name, date, and weather condition in a clean and minimal interface.

The project demonstrates the use of HTML, CSS, JavaScript, and JSON to build an interactive frontend application.

## 🚀 Features

- Search weather by city name
- Display temperature and weather condition
- Dynamic content updates using JavaScript
- Weather data stored in JSON format
- Clean and minimal UI design

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- JSON
